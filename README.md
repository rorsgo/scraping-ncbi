### **Requirements**

- [Node.js](https://nodejs.org/en/)
- [Cheerio](https://github.com/cheeriojs/cheerio)
- [got](https://github.com/sindresorhus/got)

### **Download**

```
git clone https://github.com/rorvts/scraping-ncbi.git
```

### **Install**
##### * Download dependences
```
npm install
```
##### * Executing the project
```
npm scraper.js
```
